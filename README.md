LASTEST VERSION
===============

Lastest configuration files can be found at: [env][]


INSTALLATION
============

This project has several dependencies:

- TMux
- VIM
- Zsh

To install the configuration files, follow these steps:

    $ ./install.sh


UNINSTALLATION
==============

To remove the configuration files, execute:

    $ ./uninstall.sh
